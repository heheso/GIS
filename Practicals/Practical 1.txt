GIS Practical 1 Creating and Managing Vector Data
Open QGIS and select Project -> New  (save the file too)
======================================================================================================
A) Adding vector layer, Setting properties, Vector Layer Formatting
======================================================================================================
Creating vector Layers
	a. Polygon layers (We have taken 2 layers Matunga, Garden) 
	b. Line layers (We have taken 3 layers Small_Roads, Road, Flyover)  
	c. Point layers (We have taken 3 layers bank, college, restaurants) 

1. Go to Layer -> Create Layer -> New Shapefile Layer
	For file name,  click the three dots ... to choose a file name and save location (saved as .shp file)
	Set Geometry type to Polygon/LineString/Point
	Set Coordinate Reference System (CRS) to WGS 84 (EPSG:4326)   ....if not there, then click the CRS button and select WGS 84 (EPSG:4326), then click OK
	Add Attributes (Fields): in the "New Field" section, enter a Name (e.g., "Name"), select Type (e.g., Text data), specify Length, and click Add to Fields List
	click ok to create layer
2. Plotting Features
	Select the desired layer in the Layer Panel
	Click the Toggle Editing (pencil icon) button
		For Polygons: Click Add Polygon Feature, click at least 3 points on the map, and right-click to finish
		For Lines: Click Add Line Feature, click at least 2 points, and right-click to finish
		For Points: Click Add Point Feature, click at least 1 point
	Once feature added, enter the ID and Name in the attribute window, then click Ok
3. Setting properties
	Double click the layer or right-click the layer and select Properties -> symbology tab
	Choose a color, pattern, or line width
	For label, right-click the layer and select Show labels

======================================================================================================
B) Calculating Line Lengths and Statistics
======================================================================================================
1. Go to Layer -> Add Layer -> Add Vector Layer
	For Vector Dataset, Browse and add the IND_rails.shp and IND_adm0.shp files
	set properties as you wish, for IND_adm0, outline is recommended and decrease width while for IND_rails choose dark cpolor and decrease width
2. Right click on IND_rails and select open Attribute Table
3. Open Field Calculator(fourth last, you will get name by hovering it)
	Set output field name to Track_Len
	Set output field type to decimal number
	Set precision to 2
	Set expression to $length / 1000
	click ok
4. Go to Vector -> Analysis Tools -> Basic Statistics for Fields
	Select IND_rails as the Input layer and Track_Len as the field
	Click run
5. Open the resulting output.html(i.e. in the log tab) to find the Sum, which represents the total length of railway tracks (e.g., 60,479.32 KM)
