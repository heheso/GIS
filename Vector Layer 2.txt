================================================
Step 1 — Add a Highway
================================================
Go to Layer → Create Layer → New Shapefile Layer

Geometry Type: Line

Add field:
id (Text)

Draw one long road across the map.

Enter attribute:
id
375

================================================
Step 2 — Add 6 Farms
================================================
Farms are polygons.

Create Layer

Geometry: Polygon

Fields:
Field		Type
id		Integer
name	Text

Example:
id	name
1	farm 1
2	farm 2
3	farm 3
4	farm 4
5	farm 5
6	farm 6

⚠ Ensure 2 farms are near the highway.
